Tic Tac Toe en Ligne avec IA

Une version en ligne du célèbre jeu Tic Tac Toe (morpion), où vous pouvez jouer à deux joueurs ou contre une Intelligence Artificielle (IA) de différents niveaux.

Description du jeu

Le jeu se déroule sur un plateau de 3x3 cases. Deux joueurs placent à tour de rôle leurs symboles :

X pour le premier joueur

O pour le second joueur ou l’IA

Le premier joueur qui aligne trois symboles horizontalement, verticalement ou diagonalement gagne. Si toutes les cases sont remplies sans alignement, c’est un match nul.

Niveaux d’IA

Le dépôt contient trois fichiers pour différents niveaux d’IA :

Fichier	Description
main.py	IA simple : choisit un coup de manière basique.
main1.py	IA intelligente : prend de meilleures décisions.
main2.py	IA avec Minimax : joue de façon optimale.

Vous pouvez choisir le fichier selon le niveau d’adversaire souhaité.
